자바의 기본형 데이터타입 :
	정수 : byte, short, int, long
	실수 : float, double
	문자형: char ''
	논리형: boolean

자바 스크립트의 기본형 :
	number, string:("", ''), boolean, null, undefind: 정의된 적이 없음.
	새로 추가됨: symbol: 중복되지 않는 고유값 사용
				bigint: number 타입에 안정적으로 저장할 수 없는 큰 정수 값 (number 타입의 정수뒤에 n을 붙여서 사용 혹은 함수() 사용)

타입 확인 연산자: typeof
